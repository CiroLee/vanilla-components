API

html属性 js属性 类型 说明 必填 默认值
color color primary | success | warning | danger 按钮的颜色 否 primary
variant variant solid | outline | ghost 按钮的变体 否 solid
size size sm | md | lg 按钮的尺寸 否 md
rounded rounded sm | md | lg | full | none 按钮的圆角 否 md
disabled disabled boolean 禁用状态 否 false
loading loading boolean 按钮是否处于加载状态 否 false
block block boolean 按钮是否为块级按钮 否 false
as-icon asIcon boolean 是否为图标按钮 否 false
